# Recommender System 

## Project Overview
- Developed a **Recommendation System** using **Python** and **FastAPI** to generate personalized suggestions based on user preferences.  
- Built a simple web application where users can select interests and receive relevant recommendations.  

## Features
- Personalized recommendations based on user-selected preferences.  
- Category-based recommendation generation.  
- Fast API integration for backend processing.  
- Simple and interactive user interface.  

## Categories Included
- Movies  
- Books  
- Travelling  

## Key Tasks Completed
- Designed a web interface for user input and recommendation display.  
- Implemented **logic-based filtering** for preference analysis.  
- Applied **matching algorithms** to rank recommendations using scoring techniques.  
- Connected frontend with backend APIs for dynamic response generation.  

## Technologies Used
- Python  
- FastAPI  
- HTML  
- CSS  
- JavaScript  

## Workflow
1. User selects preferences from available categories.  
2. System processes selected interests.  
3. Matching algorithm calculates recommendation scores.  
4. Personalized recommendations are displayed to the user.  

## Future Improvements
- Add database integration for larger datasets.  
- Implement collaborative filtering techniques.  
- Improve recommendation accuracy using machine learning models.
